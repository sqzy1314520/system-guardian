# Contributing

PRs welcome. This skill is about keeping Hermes healthy.

## What helps

- **New pitfalls** from your own Hermes experience
- **Translation fixes** — README is bilingual, help keep it that way
- **Reference improvements** — scripts, check procedures, build guides
- **Validation of the 12-dimension check** — does it work for your setup?

## What to avoid

- Don't inline the full SKILL.md — keep references in `references/`
- Don't add platform-specific instructions without a note
- Test on a Hermes system before submitting
