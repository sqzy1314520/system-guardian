# Hermes Agent 虽强但容易崩？我写了个开源工具，一行命令搞定系统治理

> 一个 skill，三个模式，从零搭建到每日巡检，把你的 AI Agent 管得明明白白

---

## 先说说我遇到了什么问题

如果你在用 Hermes Agent（或者任何自主 AI Agent），你一定经历过这些：

- 装完不知道从哪开始配，对着文档发呆
- Agent 跑着跑着突然「失忆」了——配置文件被冲走了
- 想检查系统健康，没有统一入口，东翻西找
- 出了 bug 不知道是配置漂移还是脚本坏了

**AI Agent 越智能，越需要治理。**

这是我的亲身经历。所以我写了一个开源工具——**System Guardian（系统卫士）**。

## Hermes Agent 是什么？为什么要管？

先简单科普。**Hermes Agent** 是 Nous Research 开源的自主 AI Agent 框架，能帮你：

- 用自然语言驱动终端操作
- 自动执行多步骤工作流
- 读写文件、搜索网络、管理进程
- 支持多种 AI 模型后端

但自主能力越强，**失控风险越大**。一个错误的 prompt 可能导致文件被覆盖，一个配置漂移可能让整个系统静默失效。

这就是 System Guardian 诞生的原因。

## System Guardian 能做什么？

一句话概括：**「帮我建系统」「检查一下」「查查哪里不对」「我能做什么」——四个命令，覆盖 Hermes 治理全生命周期。**

### 🏗️ 模式一：系统构建（免费）

从零搭建 Hermes 治理体系，三个问题搞定：

```
你：帮我建系统
Agent：怎么称呼你？
你：小智
Agent：主要用来做什么？
   [1] 工作  [2] 自媒体  [3] 学习  [4] 都有
你：4
Agent：还需要管其他事吗？
你：不需要了
```

然后自动完成：

- ✅ 控制台就位
- ✅ 宪法（SOUL.md）写入
- ✅ 红线划定（6条不可逾越的边界）
- ✅ 场景划分（工作/学习/创作）
- ✅ 内容锚定（系统保险丝）
- ✅ 开机自检配置

### ✅ 模式二：系统自查（免费）

每日健康检查，一个命令 3 秒出结果：

```bash
你：检查一下
```

![自查演示](https://raw.githubusercontent.com/sqzy1314520/system-guardian/main/screenshot-selfcheck.png)

*10 项检查，3 秒跑完。身份合规、版本一致性、脚本完整性、记忆水位、内容锚点……*

你不需要是运维专家——每个检查项都告诉你「哪里有问题、怎么修」。

### 🔍 模式三：系统审计（完整版）

当你想深度检查系统架构健康度时——

6 维度探针：

| 维度 | 检查什么 |
|------|---------|
| 约束管道 | 红线有没有真正拦住危险操作？ |
| 控制治理 | 场景之间有没有互相越界？ |
| 监督检查 | 日志、告警、审计链是否完整？ |
| 协调路由 | 消息路由是否正确分流？ |
| 调度执行 | cron job 是否按时跑？ |
| 基础完整性 | 核心文件有没有缺失或冲突？ |

输出三幕法报告：**肯定 → 否定（具体攻击点）→ 再肯定（修复方向）**。

### 📋 模式四：系统能力（免费）

一目了然看你的 Agent 能做什么、不能做什么：

```
╔══════════════════════════════════╗
║         Capability Matrix        ║
╠══════════════════════════════════╣
║ 👁️  See     ████████░░  80%     ║
║ 👂  Hear    ████████░░  80%     ║
║ 📖  Read    ██████████ 100%     ║
║ 🗣️  Speak   ██████████ 100%     ║
║ 🎨  Create  ██████░░░░  60%     ║
╚══════════════════════════════════╝
```

比如你想让 Agent 能「听」——一看就知道少了语音模块，直接去配就行。

## 如何安装？

一行命令安装：

```bash
hermes skills install https://github.com/sqzy1314520/system-guardian/raw/main/SKILL.md
```

然后用自然语言跟它对话即可：

| 你说 | 它做什么 |
|------|---------|
| "帮我建系统" | 三问题自动搭建 |
| "检查一下" | 10 项健康检查 |
| "查查哪里不对" | 深度架构审计 |
| "我能做什么" | 能力矩阵展示 |
