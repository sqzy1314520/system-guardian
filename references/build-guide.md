# 构建模式 · 固化+选择式引导

## 固化内容（用户不需要选，系统自动添加）

| 模块 | 说明 |
|------|------|
| 控制台 | control profile + 定位声明 |
| 全球认知声明 | 4条固定模板 |
| 行为底线 | 3条固定模板 |
| 6条红线 | 不可逾越的边界 |
| 内容锚定 | content-anchors.md + compliance-check.sh |
| 开机自检 | .bashrc触发 + control启动指令 |
| 审批配置 | approvals.mode: manual |

## 用户需要选择的

| 问题 | 选项 |
|------|------|
| ① 怎么称呼你？ | 用户输入 |
| ② 主要用来做什么？ | [1]工作 [2]自媒体 [3]学习 [4]都有 [5]其他 |
| ③ 还需要管其他事吗？ | 根据第②步动态生成 |

## 交互流程

```
选[1] 🏗️ 系统构建
  ↓
① 怎么称呼你？
② 主要用来做什么？
③ 还需要管其他事吗？
  ↓ 确认
正在搭建——
· 控制台就位（总管家）
· 宪法写入 ✓
· 红线划定 ✓
· 工作区建立 ✓
· 内容保险 ✓
· 开机自检 ✓
· 全部就绪 ✓
  ↓
全部完成！试试说"查看能力"看看系统能做什么。
```
